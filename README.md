# CNRoutingAssignment

## Computer Networks - Routing Configuration Assignment

This repository contains the practical implementation of network topologies and routing mechanisms using Cisco Packet Tracer. The project evaluates three structural methods of network configurations to establish successful end-to-end communication across remote local area segments.

### Contained Network Implementations:
1. **Static Routing Configuration (`.pkt` file)**: Establishment of manual next-hop static pathways across the enterprise campus model.
2. **RIPv2 Dynamic Routing Configuration (`.pkt` file)**: Automated distance-vector classless routing implementation with disabled auto-summarization properties.
3. **EIGRP Dynamic Routing Configuration (`.pkt` file)**: High-speed convergence deployment bound under Autonomous System 10 utilizing the dynamic DUAL engine algorithm.

### Verification Matrix:
All configurations have been fully converged and validated using ICMP terminal ping verifications showing a stable 0% packet loss status across all subnets.
